Wiki - Installer C-File Browser

Sur cette page; nous allons voir comment installer c-file browser sur votre serveur web.

Pour commencer, téléchargez c-file browser; c'est simple; rendez vous a l'Acceuil et choisissez en quel type vous souhaitez le télécharger.
Une fois cela fait, extrayez l'archive téléchargée dans un des répertoires de votre serveur web.
Ensuite, ouvrez le fichier settings.php; choisissez la langue de votre choix (fr pour français et en pour englais) puis choisissez le dossier d'index.

C'est bon ! C-File Browser est prêt à être utilisé !
